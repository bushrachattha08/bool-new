# bool-new
Basic Features:
* Manage posts and media
* Categorize posts
* User Roles
* Content moderation
* Markdown Editor
* paginatation
* and more...

Installation
Download this repo.
Rename .env.example to .env and fill the options.
Run the following commands:

composer install

npm install

php artisan key:generate

php artisan migrate

php artisan db:seed

php artisan serve
